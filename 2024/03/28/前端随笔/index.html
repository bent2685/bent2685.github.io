Javascript

两个对象间变量值的追加和覆盖

  • 覆盖并追加
const foo = { a: 1, b: 2 }
const bar = { b: 3, c: 4}

// -> 1
Object.keys(bar).forEach(key => {
  foo[key] = bar[key]
})

// or 2
const foo2 = {...foo, ...bar}

console.log(foo) // { a:1, b:3, c:4 }
console.log(foo2) // { a:1, b:3, c:4 }
  • 仅覆盖
const foo = { a: 1, b: 2 }
const bar = { b: 3, c: 4}

Object.keys(foo).forEach(key => {
  if (!(key in bar)) return
  foo[key] = bar[key]
})

console.log(foo) // { a:1, b:3 }

CSS

滚动字幕

给需要滚动的字幕设置display: inline-block;这样就可以通过100%相对量拿到文本的相对宽度

<template>
	<div class="container">
  	<p class="scrolling-text">滚动字幕~~滚动字~~滚动~~滚！！！</p>
	</div>
</template>

<style>
.container {
  width: 300px;
  overflow: hidden;
}
.scrolling-text {
  display: inline-block;
  white-space: nowrap;
  animation: scroll-h 10s linear infinite;
}
  
@keyframes scroll-h {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(calc(-100% + 300px));
  }
}
</style>
